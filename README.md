# ITI ICC Full-Stack JavaScript Development Program Tasks

This repository documents the tasks and projects completed as part of the **Information Technology Institute (ITI) Intensive Code Camp (ICC) Full-Stack JavaScript Development Program**. The curriculum is designed to build a comprehensive skill set in modern web development, covering foundational IT concepts, the MEARN stack (MongoDB, Express.js, Angular/React, Node.js), advanced JavaScript topics, and essential professional skills.

## Getting Started

To run the projects in this repository, you will typically need the following installed on your system:

*   **Node.js** (LTS version recommended)
*   **npm** or **yarn** (package manager)
*   **MongoDB** (or access to a MongoDB Atlas instance)
*   **Git** (for source control)
